# Preprocess the eset

Preprocess the eset

## Usage

``` r
preprocess_eset(eset, dbs)
```

## Arguments

- eset:

  the input eset typically the raw gene expression matrix

- dbs:

  the list of databases in the list of lists that can be called from the
  package's internal data

## Value

The filtered expression set that contains 0 variability ligands

## Examples

``` r
if (FALSE) { # \dontrun{
preprocess_eset(eset, dbs)
} # }
```
